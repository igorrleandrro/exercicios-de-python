# 🎲 Remoção de Elementos Aleatórios em Listas (Python)

Este projeto foi desenvolvido para praticar a manipulação de listas e geração de dados dinâmicos. O programa gera uma lista com N números inteiros aleatórios entre 1 e 10, solicita um número X ao usuário e elimina todas as ocorrências de X da lista utilizando a função `del`.

## 🧠 O que foi praticado neste exercício:
* **Geração de Dados Aleatórios:** Uso da biblioteca nativa `random` com a função `randint`.
* **Busca e Exclusão Dinâmica:** Varredura de listas utilizando o loop `while` combinado com controle rigoroso de índices.
* **Uso da função `del`:** Manipulação direta da estrutura da lista para remoção de elementos por índice sem quebrar a iteração.
