# Remoção de Elementos Duplicados em Listas (Python)

Este projeto foi desenvolvido para resolver um problema clássico de estrutura de dados: ler uma lista de números inteiros fornecida pelo usuário e eliminar os elementos repetidos, mantendo estritamente a primeira ocorrência de cada valor.

## O que foi praticado neste exercício:
* **Manipulação de Listas:** Preenchimento dinâmico e controle de dimensões de arrays.
* **Estruturas de Repetição:** Uso de loops encadeados para varredura de índices.
* **Lógica de Ponteiros:** Utilização de variáveis de controle manual (`m` e `j`) para comparação de elementos.
* **Otimização de Algoritmos:** Análise de complexidade e evolução de código para abordagens mais eficientes.
